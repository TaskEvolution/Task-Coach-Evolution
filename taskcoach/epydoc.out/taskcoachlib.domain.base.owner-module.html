DomainObjectOwnerMetaclass(name, bases, ns)

source code 

This metaclass makes a class an owner for some domain objects. The __ownedType__ attribute of the class must be a string. For each type, the following methods will be added to the class (here assuming a type of 'Foo'):

  • __init__, __getstate__, __setstate__, __getcopystate__, __setcopystate__
  • addFoo, removeFoo, addFoos, removeFoos
  • setFoos, foos
  • foosChangedEventType
  • fooAddedEventType
  • fooRemovedEventType
  • modificationEventTypes
  • __notifyObservers
