Class AutoImporterExporter

source code

object --+
         |
        AutoImporterExporter

AutoImporterExporter observes task files. If a task file is saved, either by the user or automatically (when autosave is on) and auto import and/or export is on, AutoImporterExporter imports and/or exports the task file.
