Class wxReportScheduler

source code

wx.Printout --+

This is a class which demonstrate how to use the in-memory wxSchedulerPrint() object on wxPython printing framework. You can control wxScheduler in the same way on GUI. For other info on wxPrintOut class and methods check the wxPython documentation (RTFM for nerds ;-) ).
